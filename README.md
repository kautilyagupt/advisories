# Marion & Cactus Jump Fusion Game

A browser-based fusion of Marion (Donkey Kong-style) and Cactus jump game mechanics.

## Features
- Platform jumping gameplay
- Obstacle avoidance
- Score system
- Lives system

## How to Play
Open `index.html` in a browser to start playing!

## Tic Tac Toe Game
The repository now also includes a Tic Tac Toe web game. Open `index.html` in your browser to play.

## GitHub Pages
If you want to share the game as a browser link, enable GitHub Pages for this repository and use the link:

`https://kautilyagupt.github.io/advisories/`
